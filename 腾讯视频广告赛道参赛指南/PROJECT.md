# EmotionSense Ad · 详细产品文档

> 情绪智投 — 基于多模态情绪感知与强化学习的智能广告决策引擎

---

## 1. 产品定位

**一句话定位**：将 AI 多模态情绪感知能力引入视频平台广告投放链路，实现从"强制曝光"到"精准共情"的范式转换——广告不是打断用户，而是在用户愿意倾听的时刻轻轻走进他们心里。

**核心洞察**：传统广告的"打断"逻辑与用户情绪需求从根本上对立——用户抗拒的不是广告本身，而是广告出现在错误的时刻。情绪共情是打破这个困局的唯一钥匙。

**差异化价值**：现有视频平台（爱奇艺、优酷、YouTube）的广告策略均基于"人群定向"或"跳过机制"，没有一个真正实现了"情绪时机感知"。情绪智投是首个将 AI 情绪感知 + 强化学习决策 + 隐私保护三位一体落地的方案。

---

## 2. 产品设计核心逻辑

情绪智投技术架构分为四个层次，从底层的感知捕获到顶层的持续进化，形成完整的"感知 → 决策 → 保障 → 进化"闭环系统。

**产品设计核心逻辑**：先通过多模态信号（面部微表情 + 视频播放行为 + 凝视轨迹）将用户精准划分为高/低情感需求两类群体，高情感需求用户处于深度沉浸状态时绝对禁止广告插入，低情感需求用户则可开启标准化广告窗口。在此基础上，AI Agent 以 ε-greedy Q-Learning 驱动，实时学习不同情绪状态下的最优广告策略，实现从"强制曝光"到"精准共情"的范式升级。

---

## 3. 用户分层策略

### 高情感需求用户（核心目标人群）

- **代表场景**：追情感剧、悬疑剧、烧脑剧，需要集中注意力共情
- **心理**：看剧是一种情绪体验和情感补偿，不想被广告打断心流
- **特征**：追求情感代入感，沉浸其中，不希望被打断
- **策略**：精准捕捉其情绪低谷窗口，绝对禁止在高情感投入时插入广告

### 低情感需求用户（次要触达人群）

- **代表场景**：通勤、做家务、睡前作为背景音，不追求深度沉浸
- **心理**：对剧情投入度低，广告打断对观感影响小，是标准化广告投放的基础流量池
- **特征**：观看注意力分散，可接受打断，广告填充率高
- **策略**：可开启标准化广告窗口，保证平台整体广告填充率

## 4. 个性化引擎

### 为什么需要个性化？

通用模型只告诉你"用户现在开心"，但不会告诉你"用户**比平时**更开心"。情绪智投的个性化引擎为每位用户建立独立的情绪基线，所有后续决策都基于个人偏差，而非固定阈值。

### 核心流程

**Step 01 · 个人情绪基线建立**：首次使用时，AI 在 30 秒内通过面部微表情建立个人情绪基准——开心时嘴角上扬幅度、平静时眉心舒展程度。每个人都是独特的。

**Step 02 · 实时偏差追踪**：每次检测时，将当前情绪与个人基线对比，计算情绪偏移量。偏移方向比绝对值更重要——你正在变得更开心还是更平静？

**Step 03 · 动态策略适配**：Q-Learning Agent 基于个人情绪轨迹动态调整策略——你的偏好会随着观剧进度变化，广告策略也随之适应，而非机械重复。

**Step 04 · 自我迭代优化**：每当你跳过广告，Agent 记录"负反馈"；每当你不排斥广告，Agent 记住"正反馈"。越用越懂你，广告打扰越来越少。

### 个性化决策面板说明

| 模块 | 说明 |
|------|------|
| 个人策略更新次数 | 你的 Q 表随反馈迭代的次数，越多越精准 |
| 探索率 ε | Agent 尝试新策略的比例，随学习逐步降低 |
| 个人累计奖励 | 你的正反馈累积值，代表广告接受度 |
| 个人情绪基线 | 首次 30 秒采集，建立你独有的表情基准档案 |
| 动态策略适配 | 基于你的行为反馈自动调整，越用越精准 |
| 隐私隔离 | 个人 Q 表参数经差分隐私噪声处理后上传聚合 |
| 行为信号融合 | 融合暂停/视线比/离开频率，扩展个性化决策维度 |

## 5. 多模态 AI 检测

多模态 AI 检测不止于表情。

- **面部微表情**：眉心舒展度、嘴角弧度、眼睑运动，识别 6 种基础情绪状态
- **视频播放行为**：暂停次数、拖拽频率、观看时长，自动推断情感投入深度
- **凝视轨迹**：眼睛位置、眨眼频率、视线离开屏幕时长，判断注意力是否集中
- **历史偏好数据**：同类型剧集历史情绪曲线，建立个人情绪基线，提升检测准确率

---

## 6. 技术路线图

- **感知层（多模态情绪融合）**：面部微表情 + 视频播放行为 + 凝视轨迹 + 历史偏好
- **决策层（DQN 强化学习）**：ε-greedy 探索 + 动态 exploitation，避免局部最优
- **保障层（边缘 AI 隐私保护）**：本地计算 + 差分隐私 + 联邦学习
- **进化层（数据闭环自迭代）**：IndexedDB 存储 → CSV/JSON 导出 → TensorFlow/PyTorch 训练

---

## 7. 落地规划（两阶段推进）

### Phase 01 · 前期：保证播放量

优先保证平台整体播放量不受影响。通过情绪识别找到"绝对平静"的窗口精准插入广告，核心原则是"宁可少投，不可错投"。同步采集多模态行为数据（暂停次数、凝视时长、视线离开频率等），为 Phase 02 打下数据基础。

### Phase 02 · 后期：情绪智投

在积累足够的用户行为数据后，升级为多模态 AI 决策引擎：融合面部情绪、行为信号、凝视轨迹，精准识别高情感需求用户，在其情绪低谷窗口实现广告价值最大化。AI Agent 自主学习最优策略，替代人工配置。

---

## 8. 隐私合规

用户对摄像头采集面部数据天然存在隐私顾虑——边缘 AI 本地计算 + 差分隐私技术让隐私保护成为产品特性而非限制。

- **本地计算**：所有 AI 推理在手机端完成，原始数据不出设备
- **差分隐私**：上传标签添加数学噪声，防止个体被识别
- **联邦学习**：多用户数据聚合训练，单一用户数据不泄露
- **隐私隔离**：个人 Q 表参数经差分隐私噪声处理后上传聚合，服务器无法还原个体策略
